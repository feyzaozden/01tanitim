# Kişisel Tanıtım Sayfası

Frontend öğrenme sürecimde HTML temellerini uygulamak amacıyla geliştirdiğim kişisel tanıtım sayfasıdır.

## Projenin amacı

Bu projede framework kullanmadan doğru ve erişilebilir bir HTML yapısı oluşturmayı hedefledim.

## Kullanılan teknolojiler

- HTML5
- Git
- GitHub

## Sayfa bölümleri

- Hakkımda
- Becerilerim
- Projelerim
- Öğrenme sıram
- İletişim formu
- Faydalı bağlantılar

## Öğrendiğim konular

- HTML belge yapısı
- Başlık ve içerik hiyerarşisi
- Semantik HTML etiketleri
- Liste ve bağlantılar
- Görseller ve alt metni
- Form alanları
- Temel HTML doğrulaması
- Erişilebilir HTML
- Git commit işlemleri

## Projeyi çalıştırma

1. Proje dosyalarını bilgisayarınıza indirin.
2. Proje klasörünü VS Code ile açın.
3. `index.html` dosyasını Live Server ile çalıştırın.
